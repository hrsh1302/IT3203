Evolution of World Wide Web (WWW)

World Wide Web, a revolution

Have you ever though of the impact the World Wide Web has in our daily life? Personally, there isn't a single day in which I don't consult the WWW. Using my smartphone to scroll over the Internet, or working on my computer requires access to the WWW.

Key Terms

Term What it Means
The World Wide Web A system of web pages and resources accessed via the Internet
The Internet A global network of interconnected computers enabling communication and data exchange
HTTP (Hypertext Transfer Protocol) A protocol for transferring web data
URL (Uniform Resource Locator) The address of a web resource
